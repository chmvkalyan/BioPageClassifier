Presentation Chris Armstrong Armstrong Process Group Presentation at SATURN 2014 The speaker presents best practices for using reference models for categorizing architecture content Publisher Software Engineering Institute Subjects Situational AwarenessSituational Awareness Watch Abstract The notion of reference models and reference architectures are used pervasively in the architecture profession However when applying these concepts in practice there is often confusion as to how they are different yet related The speaker will present best practices regarding how to use reference models for categorizing architecture content for different purposes such as planning management gap analysis and alignment and how they relate to architecture and solution building blocks per TOGAF The presentation continues with a discussion of how reference architectures are similarly used but also how they can be used to represent architecture patterns design patterns and best practices Share
